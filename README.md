
***EXPENSE TRACKER***

## OVERVIEW….

Expense Tracker is a simple Python project that helps users record and manage their daily expenses. It allows users to store expense details in a CSV file, making it easy to keep track of spending habits. This project is beginner-friendly and demonstrates Python concepts such as file handling, functions, loops, conditional statements, and CSV operations.

## FEATURES…

- Add Expense
- View Expenses
- Search Expense
- Delete Expense
- Total Expense
- Monthly Reports

## FEATURES EXPLANATION…

- Add new expenses with date, category, amount, and description.

- View all saved expenses in an organized format.

- Search expenses by category.

- Delete unwanted expense records.

- Calculate the total amount of all recorded expenses.

- Generate a monthly expense report.

- Automatically creates the CSV file if it does not exist.

- Simple menu-driven interface for easy navigation.

- Stores data permanently using CSV files.

## TECHNOLOGIES USED…

- Python
- CSV Module
- OS Module
- File Handling
